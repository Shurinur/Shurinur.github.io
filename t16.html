CSS w stronach

CSS - Cascading Style Sheet, czyli inaczej kaskadowe arkusze stylów. Jeśli chcemy by nasza strona wyglądała choć troche tak jak w naaszej wyobraźni, a nie jak w latach 90' to trzeba bedzie je wykorzystać, gdyż dzieki nim możemy dokładniej dopracować proste wizualne części strony, takie jak czcionka, grubość lini w tabelach lub też bardziej skomplikowane tj. animacje czy zmiany zachowania tekstu podczas najechania na niego kursorem.

Przy większych stronach najlepiej będzie jeśli do naszego "index.html" podłączymy osobny arkusz stylów, oszczędza nam to masę czasu. W CSS posługujemy się:

  • Selektorami - Czyli znacznikami, dzięki którym interpreter odnajdzie się w kodzie HTML.
  • Nawiasami klamrowymi { } - Wytępują zaraz po selektorze, interpreter traktuje je jak otwarcie/zamknięcie bloku z kodem do danego selektora/li>
  • Właściwości - Określają co chcemy zmienić w wyglądzie naszej strony
  • Wartości właściwości -Występują po właściwości, poprzedzone są dwukropkiem, i zakończone średnikiem
Przykład
  Selektor{
      Właściwość: Wartość;
  }
